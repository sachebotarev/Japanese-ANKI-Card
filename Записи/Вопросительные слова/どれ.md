# どれ

どれ

который; какой из них

![[Произношение/Вопросительные слова/どれ_слово.mp3]]

![[Картинки/Вопросительные слова/どれ.png]]

## Примеры

### Пример 1

![[Произношение/Вопросительные слова/どれ_пример_1.mp3]]

**どれ**があなたのかばんですか。

Которая сумка ваша?

### Пример 2

![[Произношение/Вопросительные слова/どれ_пример_2.mp3]]

この中で**どれ**が一番安いですか。

Какая из них самая дешёвая?

### Пример 3

![[Произношение/Вопросительные слова/どれ_пример_3.mp3]]

店に同じような傘がたくさんあったので、自分の傘が**どれ**か分からなくなりました。

В магазине было много похожих зонтов, поэтому я перестал понимать, какой из них мой.

## Заметки

どれ используется самостоятельно: «который из нескольких?». Если после него идёт существительное, нужно どの.

## Кандзи-разбор

Обычно пишется хираганой. Кандзи-разбора нет.
