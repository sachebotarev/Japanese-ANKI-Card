# 降りる

おりる

выходить (из транспорта); спускаться

![[Произношение/Глаголы/降りる_слово.mp3]]

![[Картинки/Глаголы/降りる.jpg]]

## Примеры

### Пример 1

![[Произношение/Глаголы/降りる_пример_1.mp3]]

バスから**降ります**。

Выхожу из автобуса.

### Пример 2

![[Произношение/Глаголы/降りる_пример_2.mp3]]

次の駅で電車を**降りました**。

Я вышел из поезда на следующей станции.

### Пример 3

![[Произношение/Глаголы/降りる_пример_3.mp3]]

山の頂上に着いてから、ゆっくり**降りて**きました。

Добравшись до вершины горы, я не торопясь спустился вниз.

## Заметки

降りる — «выходить из транспорта/спускаться» (непереходный). Часто: транспорт + を (電車を降りる).

## Кандзи-разбор

降 (コウ, お.りる, ふ.る) — «спускаться; падать».
