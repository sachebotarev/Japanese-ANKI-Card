# 会う

あう

встречаться; встречать кого-то

![[Произношение/Глаголы/会う_слово.mp3]]

![[Картинки/Глаголы/会う.png]]

## Примеры

### Пример 1

![[Произношение/Глаголы/会う_пример_1.mp3]]

駅で友達に**会います**。

Я встречусь с другом на станции.

### Пример 2

![[Произношение/Глаголы/会う_пример_2.mp3]]

明日、図書館で先生に**会う**予定です。

Завтра я планирую встретиться с учителем в библиотеке.

### Пример 3

![[Произношение/Глаголы/会う_пример_3.mp3]]

久しぶりに兄に**会った**ので、駅前のカフェでお茶を飲みながらたくさん話しました。

Я давно не видел брата, поэтому, встретившись с ним, много говорил с ним за чаем в кафе у станции.

## Заметки

会う обычно требует частицы に для человека, с которым встречаются: 友達に会う. Место встречи часто выражается через で: 駅で会う.

## Кандзи-разбор

会 — встреча, собрание; ключевой кандзи для ситуаций встречи людей.
