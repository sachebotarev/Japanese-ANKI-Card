# 進める

すすめる

продвигать; продвигаться; ускорять; рекомендовать

![[Произношение/Глаголы/進める_слово.mp3]]

![[Картинки/Глаголы/進める.jpg]]

## Примеры

### Пример 1

![[Произношение/Глаголы/進める_пример_1.mp3]]

仕事を**進めます**。

Продвигаю работу вперёд.

### Пример 2

![[Произношение/Глаголы/進める_пример_2.mp3]]

プロジェクトを少しずつ**進めて**います。

Постепенно продвигаю проект.

### Пример 3

![[Произношение/Глаголы/進める_пример_3.mp3]]

計画を予定通りに**進める**ために、毎日チームで話し合いをしています。

Чтобы продвигать план по расписанию, команда проводит встречи каждый день.

## Заметки

他動詞; противоположность — 進む（すすむ）= идти, продвигаться самостоятельно; 進める также может значить «советовать, рекомендовать»

## Кандзи-разбор

進(すす)める=продвигать → 進める=продвигать, рекомендовать
