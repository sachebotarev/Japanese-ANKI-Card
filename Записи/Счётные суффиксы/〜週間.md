# 〜週間

しゅうかん

счётный суффикс для недель

![[Произношение/Счётные суффиксы/〜週間_слово.mp3]]

![[Картинки/Счётные суффиксы/〜週間.png]]

## Примеры

### Пример 1

![[Произношение/Счётные суффиксы/〜週間_пример_1.mp3]]

アメリカへ一**週間**旅行します。

Едем в США на неделю.

### Пример 2

![[Произношение/Счётные суффиксы/〜週間_пример_2.mp3]]

研修は二**週間**で終わります。

Стажировка закончится за две недели.

### Пример 3

![[Произношение/Счётные суффиксы/〜週間_пример_3.mp3]]

要件のドラフト確認に三**週間**以上かかっているあいだに、仕様が二度変わってしまいました。

Пока правки черновика требовали больше трёх недель, спецификацию дважды поменяли.

## Заметки

〜週間 показывает длительность в неделях. Удобно для разговоров о курсах, отпуске, лечении и планах.

## Кандзи-разбор

週 (シュウ) — «неделя», 間 (カン, あいだ) — «промежуток». Вместе образуют счёт длительности в неделях.
