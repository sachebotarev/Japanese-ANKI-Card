# Japanese ANKI Card

Репозиторий для подготовки карточек Anki при изучении японского языка. Слова из файлов в `Слова/` превращаются в JSON-карточки, изображения и аудио, а затем синхронизируются с колодой **«Японские слова»** в Anki.

Все переводы, пояснения и учебные заметки в проекте ведутся на русском языке.

## Структура проекта

| Путь | Назначение |
|------|------------|
| `Слова/` | Исходные списки слов: один файл на тему, по одному слову на строку |
| `Карточки/<тема>/` | Готовые JSON-карточки, один файл на слово |
| `Картинки/<тема>/` | Локальные изображения для карточек |
| `Произношение/<тема>/` | Локальные MP3-файлы для слов и примеров |
| `scripts/` | Локальные утилиты проекта |
| `docs/` | Подробные правила и спецификации проекта |
| `AGENTS.md` | Короткая оперативная инструкция для ИИ-агента |

Тема в `Карточки/<тема>/`, `Картинки/<тема>/` и `Произношение/<тема>/` должна совпадать с именем файла-списка в `Слова/`.

## Краткий Workflow

1. Подготовить файл в `Слова/`: удалить пустые строки и дубли, отсортировать записи.
2. Если пользователь дал слово без указания темы, самостоятельно отнести его к наиболее подходящему файлу в `Слова/` и соответствующей папке в `Карточки/`. Новую тему создавать только когда подходящей существующей темы действительно нет.
   Исключение: `Слова/Кандзи` не пополняется автоматически. Строки в этот файл добавляет только пользователь вручную.
3. Создать или обновить JSON-карточки в `Карточки/<тема>/`.
4. Если в JSON поле `Картинка` содержит текстовый промпт и есть возможность генерации, нужно сразу создать или обновить изображение в `Картинки/<тема>/`, без дополнительных вопросов пользователю, затем заполнить этой картинкой поле `Картинка` в Anki.
5. Если в JSON поле `Произношение` пустое, сгенерировать аудио в `Произношение/<тема>/`, не перезаписывая уже заполненное поле `Произношение` в Anki.
6. Синхронизировать карточки с Anki.

## Короткие запросы

Для запуска полного пайплайна в новом диалоге достаточно короткого запроса вида:

- `Добавь слово 口座`
- `Создай карточку для слова 口座`
- `Добавь слова: 口座, 通帳, 暗証番号`

По умолчанию такой запрос означает полный цикл:

- выбрать подходящую тему;
- обновить `Слова/` там, где это разрешено правилами проекта;
- создать или обновить JSON-карточку;
- сгенерировать картинку, если в поле `Картинка` есть промпт и генерация доступна;
- импортировать картинку в Anki и заполнить поле `Картинка`;
- сгенерировать аудио, если поле `Произношение` пустое;
- создать или обновить заметку в Anki.

Если нужен не полный цикл, это лучше указать явно, например:

- `Только создай JSON для слова 口座`
- `Добавь слово 口座 без Anki`

## Требования к окружению

Для полного рабочего цикла нужны:

- установленный Anki с колодой `Японские слова` и типом заметки `Японские слова`;
- локальный `AivisSpeech`;
- `ffmpeg` для конвертации аудио;
- настроенный MCP `anki`, если синхронизация выполняется через агента.

Для озвучки по умолчанию проект предпочитает голос `Anneli-nsfw`, если он доступен в локальном `AivisSpeech`.
