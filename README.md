# Japanese ANKI Card

Репозиторий для подготовки карточек Anki при изучении японского языка: слова из списков превращаются в структурированные JSON-файлы и синхронизируются с колодой **«Японские слова»** (тип заметки **«Японские слова»**).

Переводы и пояснения в карточках — на **русском** языке.

## Структура проекта

| Путь | Назначение |
|------|------------|
| `Слова/` | Исходные списки слов: один файл на тему (например, `Слова/Профессии`), по одному слову на строку |
| `Карточки/<тема>/` | Готовые карточки: один JSON на слово, имя файла совпадает со словом (кандзи/кана) |
| `Произношение/` | Аудио к карточкам (см. ниже); при генерации голоса файлы группируются по темам, как в `Карточки/` |
| `.cursor/rules/` | Правила для ассистента: заполнение полей, синхронизация с Anki, опционально озвучка |

Тема в `Карточки/<тема>/` должна совпадать с именем файла-списка в `Слова/` (например, `Профессии` → `Карточки/Профессии/`).

## Формат JSON-карточки

Один файл — одна карточка. Типичный набор полей:

- **Слово** — словарная форма, по возможности кандзи  
- **Перевод** — общепринятый русский перевод  
- **Чтение** — хирагана или катакана  
- **Подсказка** — мнемоника без прямого указания целевого слова (без однокоренных подсказок)  
- **Пример** — фраза уровня N5 / начала N4 с грамматикой этого диапазона, не односложная  
- **Пример без слова** — тот же пример, целевое слово заменено на `_____`  
- **Пример перевод** — перевод примера на русский  
- **Картинка** — текстовый промпт для генерации иллюстрации 512×512 в японской манере по смыслу **Пример перевод**  
- **Произношение** — ссылки на аудио в Anki; в JSON при создании карточки можно оставлять пустым  
- **Заметки** — кратко, до двух предложений  
- **Кандзи-разбор** — разбор кандзи в слове  
- **Показать фуригану** — `N`, если в слове есть кандзи из объёма N4/N5, иначе `Y`  

Теги в Anki (ориентиры): для глаголов — `動詞`, для существительных — `名詞`, плюс тег, совпадающий с названием подпапки в `Карточки/` (тема).

## Рабочий процесс

1. Добавить новые строки в нужный файл в `Слова/`.  
2. Для слов без JSON в `Карточки/<тема>/` создать файлы по правилам выше.  
3. Синхронизировать с Anki через MCP-инструмент **anki** (колода и тип заметки — «Японские слова»). Перед массовым созданием имеет смысл уточнить актуальный список полей заметки через доступные инструменты Anki.

### Синхронизация с Anki (кратко)

- Новая карточка: все поля заполняются из JSON.  
- Существующая: поля обновляются из JSON, кроме особых случаев:  
  - **Произношение** не трогать, если в Anki уже заполнено.  
  - **Картинка**: не менять, если **Пример** в Anki совпадает с JSON; если пример изменился — записать новый промпт; если в Anki пусто — добавить промпт.  
- После изменений в Anki JSON в `Карточки/` дополняется недостающими полями и обновляется там, где значения в Anki изменились.

### Озвучка (опционально)

Если для карточки пустое поле **Произношение**, по правилам проекта можно сгенерировать два MP3 через ElevenLabs (голос «Shizuka - Natural and Soft»): произнесение **Слово** и **Пример**, сохранить как `<Слово>_слово.mp3` и `<Слово>_пример.mp3` в соответствующей ветке `Произношение/`, вставить ссылки в Anki и снова синхронизировать JSON.

## Инструменты

- **Anki** — MCP `anki` для создания и обновления заметок.  
- **ElevenLabs** — MCP для генерации аудио (по сценарию выше).

## Требования к окружению

Локально нужны установленный Anki с нужной колодой и типом заметки, а также настроенные MCP-серверы в Cursor (или аналог), если синхронизация и озвучка выполняются через ассистента.
